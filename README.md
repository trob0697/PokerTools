
# BombPot.com

![Website](demo.mp4)  
A website containing tools useful to pokers players.  
Tech Stack:
- React
- Python (Django)

## Getting Started

Requirements:
- Node
- Python
- pipenv

Install dependencies
```
$ cd client && npm install
```
Run client
```
$ cd client && npm start
```
Run Server
```
$ cd server
$ pipenv shell
$ python manage.py runserver
```

Access at: http://localhost:3000
